# 竞猜活动管理看板

公司每日有奖竞猜活动的管理工具，用于管理出题轮值、题库和积分榜。

## 启动

```powershell
Set-Location "C:\Users\tallm\Documents\Codes\quiz&score"
npm run dev
```

浏览器打开 http://localhost:6970

> 注意：路径含 `&` 符号，必须在 PowerShell 中运行，不能用 cmd。

## 功能

### 轮值看板
- 显示当前应出题的组，点击「已出题」将其移至队列末位
- 支持添加、编辑、删除组

### 题库
- 管理待出题目，每题包含：题目内容、题型（单选/多选）、答案、出题人、所属组
- 点击「出题」将题目移至历史出题并记录日期
- 支持一键复制题目内容

### 历史出题
- 记录已出过的题目及出题日期

### 积分榜
- 以工号为唯一标识
- 底部输入框填写姓名、工号、积分（默认 +1）提交
  - 工号已存在 → 累加积分
  - 工号不存在 → 新增人员
  - 工号存在但姓名不符 → 提交成功并弹出警告提示
- 支持编辑、删除人员

### 积分历史
- 记录每次加分的详细信息（姓名、工号、获得积分、时间）

## 数据存储

数据保存在项目根目录的 `data.json`，可直接打开查看和备份。

## 技术栈

- Vue 3 + Vite
- Pinia（状态管理）
- Element Plus（UI 组件库）
- 内置 Vite 插件提供本地文件 API（无需独立后端）
